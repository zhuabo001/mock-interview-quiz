Frontier Evo — 基金销售管理系统架构

Frontend-Only SPA · React 19 + Vite · MSW Mock · Vercel 静态部署

前端技术栈

  • • React 19 + TypeScript 6.0
  • • Vite 8 构建 + HMR
  • • React Router v7 客户端路由
  • • Ant Design v6 组件库
  • • @ant-design/charts 数据可视化
  • • Vitest + Testing Library 测试

状态与数据管理

  • • TanStack Query v5 — 服务端状态
  • • Zustand v5 — 客户端 UI 状态
  • • MSW v2 — Service Worker Mock
  • • 自定义 Hooks 封装业务逻辑
  • • 无真实后端，全量 Mock 数据
  • • VITE_USE_MOCK 环境变量控制

部署与架构特点

  • • Vercel 静态托管 (Hobby Plan)
  • • SPA Rewrite — 所有路由指向 index.html
  • • 纯前端架构，零后端依赖
  • • Service Worker 离线可用
  • • 核心实体：Fund · Client · Holding · FollowUp
  • • 设计原则：素雅专业，3 次点击可达任意详情
